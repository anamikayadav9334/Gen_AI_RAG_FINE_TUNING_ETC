# Gen_AI_RAG_FINE_TUNING_ETC

A hands-on repository for exploring Generative AI concepts including **Retrieval-Augmented Generation (RAG)**, **Fine-Tuning**, **QA with PDFs**, and **Gemini API integration**. This project contains experiments, notebooks, and a Streamlit application to demonstrate practical implementations of modern GenAI techniques.

## 🚀 Features

- **RAG (Retrieval-Augmented Generation)**: Build systems that retrieve relevant documents before generating responses
- **Fine-Tuning**: Experiments with fine-tuning language models for specific tasks
- **QA with PDF**: Extract and query information from PDF documents
- **Gemini API Integration**: Leverage Google's Gemini models via Python
- **Streamlit App**: Interactive web interface for demonstrations
- **Modular Codebase**: Reusable logging, exception handling, and templates
